Python程序后台运行

2021-07-29
其他

后台运行：使用nohup命令，nohup python3 -u run.py > log_output 2>&1 &，运行run.py文件，并将输入输出重定向，其中-u使得python不会将输出缓存。

查看进程：ps -ef | grep "python3 -u run.py"。ps会显示两个数字，第一个是进程号。

关闭进程：kill 进程号。
