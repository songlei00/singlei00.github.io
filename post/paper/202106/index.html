1. GenAttack: Pratical Black-box Attacks with Gradient-Free Optimization

使用遗传算法求解对图像分类器的黑盒定向攻击问题。

算法整体比较简单，完全是GA的框架。

  1. Fitness函数设计为增大目标类别$t$的概率，同时减小其他类别$j\neq t$的概率，并使用$\log$减少数值误差：$F(x) = \log f(x)t - \log \sum{j\neq t} f(x)_j$。
  2. 对初始图像各个像素添加均匀分布$(-\delta_{max}, \delta_{max})$的随机扰动；
  3. 选择时保留精英并对其他个体使用softmax的方式选择其他个体
  4. crossover时根据两个父代的fitness随机选择$p = \frac{fitness(p_1)}{fitness(p_1) + fitness(p_2)}$
  5. mutation为在某个小范围内随机采样并施加到每个像素上，并进行$clip(-\delta_{max}, \delta_{max})$操作

加速方法：

  1. Dimensionality Reduction：图片维度高，处理慢，所以通过在低维空间产生噪声$e_{adv} \in [0, 1]^{d'}$，通过bilinear resize的方式转化到高维空间$[0, 1]^d$，加速处理。
  2. Adaptive Parameter Scaling：对mutation的两个参数(mutation range和mutation rate)在多轮没有改进的情况下，进行衰减(退火)。

使用无梯度优化的方法攻击的优势：自然的适用于黑盒攻击；对于经过对抗训练的模型往往也会有很好的效果(一般的对抗训练都是使用白盒攻击产生对抗样本的，似乎只能防御通过类似方式产生的样本，比如使用$l_{\infty}$进行训练只能防御$l_{\infty}$产生的样本，猜的)，所以无梯度的攻击方法能够成功攻击各种对梯度做防御的网络。

防御的方法：对抗训练、gradient masking、obfuscation。

总结：主要是遗传算法的应用，以及第一种通过插值的方式从低维空间产生高维噪声的加速方式很不错。

2. ZOO: Zeroth Order Optimization Based Black-box Attacks to Deep Neural Networks without Training Substitute Models

使用梯度估计的方法解决黑盒图像样本攻击的问题，提出了ZOO算法，要求模型可以输出每一个类别的置信度。

算法思路：思想很简单，使用差分的方法，梯度等于$\frac{f(x+h) - f(x-h)}{2h}$。然后利用梯度和白盒攻击方法C&W方法生成对抗样本。

  1. C&W：问题形式化：$\min_x ||x-x_0||^2+c f(x, t), s.t. x\in [0, 1]^p$。第一项是正则化项，保证对抗样本和原数据差别不大，第二项中$t$是目标类别，$f(x, t) = \max { \max_{i\neq t}[Z(x)]_i-[Z(x)]_t, -\kappa }$，$Z(x)_i$是softmax前一层的输出结果(也许是因为进行softmax之前数值比较大，优化快吧)，$\kappa$越大，transferability越大，文中取$\kappa=0$，当第一项小于0时，表示攻击是成功的。约束条件保证对抗样本是合法的图像，可以通过令$x = \frac{1+\tanh w}{2}$，使得约束一定满足(但在一些情况下会造成梯度消失)，原问题变为无约束优化问题，可以使用梯度下降等方法求解。
  2. 黑盒攻击：将上面优化问题中黑盒设置下无法获得的项转化为可以获得的项
    1. $Z(x)i$是中间层的输出，无法获得。替换为$f(x) = \max{\max{i\neq t}\log[F(x)]_i - \log[F(x)]_t, -\kappa}$，$F$为softmax后的输出，可以获得。(可以通过简单的变化，将优化问题转化为untarget attacks)。
    2. 优化时使用的梯度信息无法获得。使用差分的方式求，$g = \frac{\partial f(x)}{\partial x_i} = \frac{f(x+he_i) - f(x-he_i)}{2h}$，$h = \frac{\partial^2 f(x)}{\partial^2 x_{ii}^2} = \frac{f(x+he_i)-2f(x)+f(x-he_i)}{h^2}$得到某个维度上的梯度，重复2p次可以得到所有维度上的梯度，但计算代价过大。使用coordinate descent的方法，每次随机挑选一个维度，计算量为2p，计算梯度$\delta^*, x_i = x_i + \delta^*$。文中对于梯度$\delta^*$的计算分别使用了Adam和牛顿法。

加速方法

  1. Dimension reduction：在低维度空间$R^m$中生成噪声，通过映射，转化到高维度$R^p$，例如通过双线性插值。
  2. Hierarchical attack：使用1的技巧后，可能无法在低维度中找到合适的噪声，所以提出了逐渐增大$m$，假设从$D: R^m\rightarrow R^p$。当loss不再下降时，增大噪声维度，通过$y = D_i^{-1}(D_{i-1}(y))$的方式，产生新的噪声(即先将低维度噪声投影到高维中，再通过新的逆映射投影到低维中)
  3. Importance sampling：使用coordinate descent时，可以选择重要的维度，优先优化。对每个区域内的像素，根据最近几轮中像素值改变的剧烈程度，赋予不同的采样概率，对变化剧烈的区域，提高概率。

实验中关注的指标是攻击成功率以及对抗样本和原图像的$l_2$距离(指示生成对抗样本的质量)。

总结：提出了通过差分的方式估计模型梯度的方法，以及加速方法，不需要训练替代模型。其中前两种加速方法似乎很常见也很好用，可以学习一下。

3. Black Box Adversarial Attacks with Limited Queries and Information

本文是对黑盒的图像分类神经网络的攻击，要求更加严格(query次数有限，不完全信息，只能获得label而不是loss函数)，使用NES算法进行梯度估计。

总结(黑盒对抗攻击)

  1. 目前图像分类中的黑盒攻击方法主要有：

    1. Gradient estimation：(通过输入差分的方式，估计网络梯度，再利用梯度按照白盒的方法求解，比如ZOO)
    2. Substitute model：(用一个网络近似，这种方法缺点比较明显，1是无法确定网络的具体结构，没有一个合适的范式，很多paper中都是使用和被攻击模型相同的网络结构，很不合理，2是攻击时利用了transferability性质，会造成精度的损失)
    3. 使用其他例如优化方法(演化算法、近似为子模问题使用贪心算法)。

  2. 使用无梯度优化的方法这个方向效果看起来还是比较好的，之前的工作比较少且比较新，并且上面的文献都集中在近一两年。图像和文本攻击都不错，图像的工作相对多一点，文本的工作更难，但这种方法应该是非常自然的可以用于文本攻击的方法，感觉更有前景。

  3. 使用演化算法求解问题时，算法框架基本是确定的，主要探究的地方还是各种加速或提升效率的trick。
