Evolutionary and Swarm Optimization

元启发式算法分为evolutionary algorithm和swarm optimization algorithm，这两种方法都是基于种群的随机搜索算法。

  1. Evolutionary algorithm的框架有genetic algorithm，evolution strategy ，genetic programming等框架。包含初始化、fitness评估、选择、重组等多个步骤。
  2. Swarm optimization algorithm中两个经典算法是particle swarm optimization和colony optimization algorithm，群智能算法通常没有重组和选择的操作。

1. Evolutionary algorithm

GA和ES的主要区别：EA中有参数的self-adaptation；ES的子代个数通常大于等于父代，GA通常相等；Gaussian mutation是变异的主要来源，GA的主要来源是crossover。ES通常用于连续优化问题，GA通常适用于整数优化和组合优化。

ES

(1+1)-ES中有一个父代，通过添加高斯扰动，生成一个子代。1/5 success rule是如果之前k轮中，成功的变异比例超过1/5则增大高斯分布的标准差$\sigma = \sigma / c, c \leq 1$，如果小于，则$\sigma = \sigma * c$，这是最简单的参数self-adaptation思想。

Environmental selection中：$(\mu, \lambda)$-ES是non-elitism的，会在$\lambda$个子代中选择$\mu$个最好的作为下一代；$(\mu+\lambda)$-ES是elitism的，会在$\mu+\lambda$个个体中选择最好的$\mu$个。

如果最优解在搜索空间的中部时，加入crossover或recombination可以带来提升。Local recombination对所有变量(即同一个染色体上的每一位变量)使用相同的父代，而global recombination则是每一位都重新随机选择两个父代。

CMA-ES算法：该算法变异时的高斯分布的协方差矩阵不是对角阵，各个变量之间是相关的，希望找到adapt协方差矩阵使得我们可以最高效的找到最优值。

Genetic Programming

GP可以看做GA使用不同的representation(通常是决策树结构表示)的变种，适用于regression和特征选择问题，因此可以看做是ML的一个子集。

Tree-based GP：GP的流程也是初始化、重组变异、评估、选择下一代，但由于表示方式的不同，所以初始化和重组变异的过程和前面的算法也有所不同。初始化有full和grow两种方法，full方法是指定一个深度，构建一个完全二叉树，而grow方法则使用DFS，某一个节点达到设定的深度就可以。重组变异是在子树级别的。

2. Swarm optimization algorithm

群智能算法和前面演化算法的区别是，群智能算法使用的信息包括了所有个体的某个信息(例如所有个体所到达的最好位置)。

Ant colony optimization

ACO是根据蚁群使用信息素交流而提出的。ACO包括四步：初始化、解构建、局部搜索、信息素更新。

Differential Evolution

DE是解决连续优化的一种方法。

初始化：第i个个体的第j维的值为$x_{j, min} + r * (x_{j, max} - x_{j, min})$，其中$r$是0到1的随机数。

Mutation：包含多种不同的变异算子。

DE/rand/1：$u_i(t+1) = x_{r1}(t) + F(x_{r2}(t)-x_{r3}(t))$

DE/best/1：$u_i(t+1) = x_{best}(t) + F(x_{r1}(t)-x_{r2}(t))$

DE/rand/2：$u_i(t+1) = x_{r1}(t) + F(x_{r2}(t)-x_{r3}(t)) + F(x_{r4}(t)-x_{r5}(t))$

DE/best/2：$u_i(t+1) = x_{best}(t) + F(x_{r1}(t)-x_{r2}(t)) + F(x_{r3}(t) - x_{r4}(t))$

DE/current-to-best/1：$u_i(t+1) = x_{i}(t) + F(x_{best}(t)-x_{i}(t)) + F(x_{r1}(t) - x_{r2}(t))$

DE/rand-to-best/1：$u_i(t+1) = x_{r1}(t) + F(x_{best}(t)-x_{r1}(t)) + F(x_{r2}(t) - x_{r3}(t))$

对于前四个，第二个指示的是base的选择，即对哪一个个体进行变异，第三个指示的是差分的数量，1表示一个差分，2表示两个差分。对于后两个，第二个指示base和差分的方向，第三个指示有随机差分的数量。DE名字应该也是从这里显示出来，通过计算个体的差分，进行mutation。

Crossover：使用Binary或exponential crossover

Particle Swarm Optimization

  1. Canonical PSO：每个个体有位置和速度，利用自身或群体的经验更新速度，再利用速度更新位置。优点是容易实现和快速收敛，缺点是容易收敛到局部最优，尤其是在高维情况下。
  2. Competitive Swarm Optimizer：缓解提前收敛的问题。将所有粒子分为两组，根据fitness值分出胜者和败者，败者向胜者学习。
  3. Social Learning Particle Swarm Optimizer：social learning是向种群中更好的个体学习。SL-PSO中，将个体按照fitness值排序，每个个体以一定概率向比他更好的个体学习

Memetic Algorithms

Memetic算法和演化算法的区别是，在子代评估之前加入了局部搜索。局部搜索有两种方式，Lamarckian和Baldwinian，区别是Lamarckian局部搜索时表现型的改变会直接改变基因型，而Baldwinian表现型的改变不会影响基因型。

多目标Memetic：局部搜索时需要使用单目标，所以通过权重向量转化单目标为多目标。随机产生一个权重向量或计算得到一个权重。

Estimation of Distribution Algorithms

EDAs是一种基于种群的算法，使用概率模型引导搜索(有点像RL中model-based的概念)。将传统的EA中的crossover和mutaiton步骤变为建立概率模型和从概率模型中采样的过程。
