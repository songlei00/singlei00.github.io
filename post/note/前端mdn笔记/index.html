学习一下网站搭建。

1. 网站组织结构

  1. 在文件名和文件中，使用中划线而不是下划线分割单词。因为搜索引擎会将中划线解析为空格，所以多个词是分开作为关键词的，而下划线则会把所有词连起来作为一个整体的关键词。
  2. 网站结构：基本的包括主页index.html、图片文件夹images、样式表文件夹styles、脚本文件夹scripts。
  3. html的作用是定义网站的内容结构(即网站各个部分位于什么位置)，css用于设置网络的样式(即让网站好看，也可以用html的属性实现简单的样式控制)，js用于与用户交互。

2. HTML基础

  1. 块级元素与内联元素：块级元素在页面中以块的形式展示，内容会出现在单独的一行中，常用于展示结构化的内容，如段落、列表、导航菜单等。内联元素通常出现在块级元素中，并包含一小部分内容，如<a>, <em>等。
  2. 空元素：只有一个标签，通常用来在元素所在位置插入一些东西。
  3. 布尔属性：没有属性值的属性，例如<input type="text" disabled>。
  4. 特殊字符：例如尖括号，引号等，需要使用特殊的编码：如&lt。
  5. 标准：元素用小写、属性值用双引号包含。
  6. 文档的基本组成部分：页眉(横跨整个页面顶部的大标题，<header>)、导航栏(<nav>)、主内容(<main>)、侧边栏(<aside>)、页脚(<footer>)。
  7. 无语义元素：<div>和<span>，<div>是块级无语义元素，<span>是内联无语义元素。

3. CSS基础

CSS用于给网页添加样式、布局、特效。

CSS中比较重要的是如何写选择器(即如何选择特定的元素)：

  1. 使用类名：在html标记中加入class="special"，然后通过.special选择。
  2. 选择特定元素的类：p.special。
  3. 包含选择器：比如选择p内的em标记：p em。
  4. 相邻选择器：与某个元素相邻的元素，如标题后的段落：h1+p。
  5. 根据状态选择：a:link和a:visited。
  6. ID选择器：#unique

加载网页的简要流程：

  1. 浏览器载入HTML。
  2. 将HTML转化为DOM。
  3. 浏览器拉取HTML相关的大部分资源。
  4. 拉取到CSS时会进行解析，渲染DOM树。
  5. 呈现在网页上。

层叠：相同优先级的规则，靠后的生效。 优先级：应用于同一个元素的多个规则，优先级最高的会显示。 继承：一些在父元素上的属性会被子元素继承。
