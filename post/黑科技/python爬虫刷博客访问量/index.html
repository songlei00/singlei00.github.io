Python爬虫刷博客访问量

2020-10-22
黑科技

使用Python爬虫刷不蒜子统计的博客访问人数和访问量。

不蒜子可以用来统计网站的访问人数和访问量，但同一个IP重复访问的话，只会增加访问量而不会增加访问人数，所以需要使用代理IP，同时刷高访问量和访问人数，代码如下。

import requests
from time import sleep

proxies = {
  'http:': 'http://127.0.0.5:10',
  'https:': 'https://127.0.0.4:11'
}
target_url = 'http://busuanzi.ibruce.info/busuanzi?jsonpCallback=BusuanziCallback_9437822328'
headers = {
  'Host': 'busuanzi.ibruce.info',
  'Origin': 'https://songlei.me',
  'Referer': 'https://songlei.me/',
  'User-Agent': 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/80.0.3987.122 Safari/537.36'
}

def get_page():
  try:
    requests.get(url=target_url, proxies=proxies, headers=headers, timeout=3)
    sleep(1)
  except Exception:
    print('error')

def main():
  for i in range(3):
    print('cnt:', i)
    get_page()

if __name__ == '__main__':
  main()

整体比较简单，如果追求高速度的话，可以使用多线程并且减少sleep时间。

其他例如CSDN等的访问量不知道可不可以刷，原理应该都是一样的。

最后还是建议删除不蒜子访问次数统计吧，加载感觉太慢了。
